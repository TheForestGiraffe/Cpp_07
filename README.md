# 42 C++ Module 07: C++ Templates

This repository contains my solutions for **42 C++ Module 07**, which
focuses on function and class templates, generic programming, and
reusable type-safe code.

## Topics Covered

-   Function templates
-   Class templates
-   Generic programming
-   Template implementation in header files
-   Dynamic memory management in template classes

## Repository Structure

``` text
Cpp_07/
├── ex00/
├── ex01/
├── ex02/
├── docs/
└── README.md
```

The assignment specification is available in the **`docs/`** directory,
where you can see exactly what was required for each exercise.

## Exercises

-   **ex00** -- Generic `swap`, `min`, and `max` function templates
-   **ex01** -- Generic `iter` function template for array traversal
-   **ex02** -- A templated `Array` class with deep copy semantics,
    bounds checking, and dynamic memory management

## Skills Demonstrated

-   Writing reusable function and class templates
-   Understanding template instantiation and compilation
-   Generic programming in C++98
-   Safe dynamic memory management
-   Deep copy semantics and exception handling

## Notes

-   Developed and tested on Linux
-   Compiled with `c++ -Wall -Wextra -Werror -std=c++98`
-   Each exercise is self-contained and can be built independently
